# Case Cracker

A single-page HTML mystery game experience built in one file.

## Files

- `casecracker.html` — the full app, including UI, styles, and game content.

## Usage

1. Open `casecracker.html` in a browser.
2. Play the game directly in the browser.

## Deploy

The project is already pushed to GitHub at:

- https://github.com/akshat-s99/casecracker

### GitHub Pages

To serve the app from GitHub Pages:

1. Go to the repository settings.
2. Under "Pages", choose the `main` branch and `/` root folder.
3. Save and wait for the site URL to become available.

## Notes

- This is a static HTML app, so no build step is required.
- The app relies on external Google Fonts and Material Icons.
